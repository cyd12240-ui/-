# 朋友局德州扑克 — 产品说明

## 产品定位

一款面向熟人圈的纯娱乐网页版德州扑克。核心是标准德扑规则 + 搞怪互动道具系统，用来替代线下的朋友桌游夜。

**一句话：** 熟人间线上延续欢乐桌游夜的德州扑克，用丢鸡蛋和送鲜花代替语言表达。

## 目标用户

- 朋友之间想联机打德州但不想用严肃扑克平台的人群
- 已经有微信群/线下牌友圈，想要线上延续娱乐氛围的人
- 不希望涉及真钱、不想要陌生人社交的人

## 解决的问题

- 现有扑克 APP 太严肃（无道具互动、无个性表达）
- 朋友间想要纯娱乐而非竞技的氛围
- 桌游局需要房间机制而非匹配陌生人
- 不想下载 APP，点链接就能进

## 第一版做什么（V1 Scope）

### 核心玩法

- [ ] 标准 52 张德州扑克，2-6 人一桌
- [ ] 初始积分 2000，小盲 10 / 大盲 20
- [ ] 完整牌局流程：Preflop → Flop → Turn → River → Showdown
- [ ] 玩家操作：Fold / Check / Call / Raise / All-in
- [ ] 边池正确计算
- [ ] 房主可选盲注自动升级

### 房间系统

- [ ] 创建房间生成 4 位数字房间码
- [ ] 通过房间码加入
- [ ] 房主开始游戏 / 踢人 / 房主转移
- [ ] 玩家列表（形象、昵称、积分、准备状态）

### 玩家形象

- [ ] 每个玩家一个卡通半身/全身形象
- [ ] 待机动画（轻微晃动、眨眼）
- [ ] 可选多个外观（动物、职业、搞笑角色）
- [ ] 被道具命中时受击动画
- [ ] 淘汰时退场动画

### 道具系统

- [ ] **丢鸡蛋**：弧线飞行 → 碎裂 → 蛋液糊脸持续到局末
- [ ] **送鲜花**：花瓣飘落飞行 → 炸开花瓣散落 → 喝彩文字
- [ ] 单次使用 + 十连发模式
- [ ] 每次使用消耗 1 积分，十连发一次扣 10
- [ ] 道具栏始终可见，每当轮到自己行动时短暂禁用
- [ ] 道具飞行全房间可见

### 界面与体验

- [ ] 卡牌滑动动画
- [ ] 积分变化数字跳动
- [ ] 行动历史追踪
- [ ] 牌型参考表（可随时打开）
- [ ] 动画可点击跳过
- [ ] 道具飞行 + 命中效果作为视觉亮点

## 第一版不做什么（Out of Scope）

- ❌ 不接入真实货币或虚拟币兑换
- ❌ 不设置聊天打字或实时语音
- ❌ 不设机器人 AI 玩家
- ❌ 不做跨房间匹配
- ❌ 不做观战模式
- ❌ 不做战绩统计或排行榜（永久排名）
- ❌ 不做道具商城或付费解锁形象
- ❌ 不做用户注册登录系统
- ❌ 不收集任何个人隐私数据
- ❌ 不嵌入广告 SDK
- ❌ 无断线重连机制（V1 暂不考虑）

## 合规调性

- 所有数值统称"积分"或"记分"——严禁出现"币""元""筹码"
- 无押注抽水机制
- 界面避免赌场视觉（绿色毡面、金边、筹码堆）
- 使用木质桌面、涂鸦牌背、彩色塑料感计分标记

## 技术约束

- 纯前端 Web 应用（HTML + CSS + JS）
- 单页面应用，无需服务端（通过浏览器本地存储或点对点通信实现游戏状态同步）
- 第一个可用版本目标：单人（单机多角色测试）/ 本地联机模式
